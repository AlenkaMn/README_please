# README_please
Краткий путеводитель по репозиториям

1. BrSTU-digital-assistant - репозиторий, содержащий базовую версию моего дипломного проекта - “Разработка интеллектуального
цифрового ассистента для поддержки студентов, сотрудников и абитуриентов
БрГТУ на основе технологии RAG и методов искусственного интеллекта”. Т.к. на данный момент я ещё не защищала дипломный проект, поэтому в общий досуп могу выложить только базовую версию своей системы.
2. EPAM_EPIC_Institute_of_Technology - я закончила международный институт от EPAM - EPIC по специальности Time Series Analysis 2023/2024. В ходе обучения было выполнено множество практических работ по предметам Practical Time Series Analysis, Data Science, Advanced Programming Languages и Machine Learning. К кажой работе также загружено исходное условие-задание.
3. AI-Masters-contest-July-2023 - в конце 1-го курса поступала в AI Masters от МГУ. В качестве вступительных испытаний было решение NP-полной задачи на укладку прямоуголников с заданными отношениями ширина:высота в заданную площадь таким образом, чтобы заполнить как можно большую площадь. В данном репозитории хранится моё решение данной задачи.
4. Neural_network_for_predicting_the_time_of_food_delivery - нейронка для предсказания времени доставки еды по возрасту, рейтингу доставщика и расстоянию от заведения до места, куда нужно доставить еду.
5. RAG-telegram-bots - мои первые RAG системы от 2024 года. Телеграм боты с ИИ, с RAG системами различных архитектур. А так же разработала подробную документацию по проектам и итоговый отчёт. Внутри есть подробные кейсы тестирования на различные поинты на домене образовательной среды.
6. Rosatom_2024 - Создание интерактивного веб-приложения для анализа данных по датасету Wine. А также в .ipynb приведён анализ датасета. Один из кейсов, решаемых в летнем студенческом ЦЕХе от Росатома.
